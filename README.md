## 解決したこと

課題1:標準入力を用いた計算機機能の実装


・問題
　計算処理を終了する事ができない。



・原因
whileループなどの繰り返し処理の場合、繰り返し処理を終了する「break」を書いていなかったから。




・解決
breakを書く事で、計算処理を終了する事ができる。
